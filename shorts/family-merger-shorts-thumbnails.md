# 썸네일 시안 3종 — 가족 제조업 합병형 가업승계 Shorts

> 익명화 원칙: 회사명·인명·지역·산업 식별자 0건. 업종/규모/상황으로만 표현.

---

## 공식 매핑 기준

검증된 3공식:
- A: 정보 비대칭 ("부자들은 알고 모르는 것")
- B: 신뢰 인물 + 구체 수치 ("연봉 10억 [직업]이 [목표] 버는 법")
- C: 손실 회피 + 타겟 명시 ("월소득 [금액] [타겟]이 반드시 알아야 하는 것")

---

## 시안 A — 정보 비대칭 공식

**제목 텍스트 (큰 글씨 2줄)**
```
부동산 많은 법인
포기 말라는 이유
```

**서브카피 (작은 글씨)**
```
컨설팅마다 틀린 말 했습니다
```

**배경 톤**: 진한 네이비(#0A1628) + 흰색 타이포
**강조 컬러**: 골드(#D4AF37) — "포기 말라는 이유" 부분만 강조
**비주얼 소재**: 법인등록증 2장이 합쳐지는 아이콘 (우측 하단)
**공식 매핑**: A (정보 비대칭 — 컨설팅이 틀렸다는 반전)

---

## 시안 B — 손실 회피 + 타겟 명시 공식

**제목 텍스트 (큰 글씨 2줄)**
```
600억 한도
그대로 쓰는 법
```

**서브카피 (작은 글씨)**
```
인적분할 없이 가능합니다 — 가족 제조업 오너 必
```

**배경 톤**: 짙은 버건디(#5C1A1A) + 흰색 타이포
**강조 컬러**: 밝은 황금색(#FFD700) — "600억" 숫자만 초대형 강조
**비주얼 소재**: 큰 숫자 "600억" 배경 워터마크 처리 + 법인 합병 화살표 아이콘
**공식 매핑**: B (구체 수치 + 타겟 명시 "가족 제조업 오너 必")

---

## 시안 C — 손실 회피 + 시간 촉박 공식

**제목 텍스트 (큰 글씨 2줄)**
```
6개월 늦으면
세금 10억 더 낸다
```

**서브카피 (작은 글씨)**
```
사전증여+합병 같은 해 안에 끝내야 하는 이유
```

**배경 톤**: 다크 차콜(#1C1C1C) + 레드 경고 포인트
**강조 컬러**: 레드(#E53E3E) — "10억 더 낸다" 강조
**비주얼 소재**: 달력 아이콘(6개월 카운트다운) + 세금 고지서 아이콘
**공식 매핑**: C (손실 회피 — "안 하면 더 낸다" 긴장감)

---

## 추천 우선순위

| 순위 | 시안 | 이유 |
|---|---|---|
| 1순위 | 시안 B | 구체 수치 "600억"이 타겟 오너에게 가장 직접적 |
| 2순위 | 시안 C | "6개월 / 10억" 손실 회피 — 후계자 자녀에게 긴박감 자극 |
| 3순위 | 시안 A | A/B 테스트 데이터 없는 경우 안전한 반전 프레임 |
